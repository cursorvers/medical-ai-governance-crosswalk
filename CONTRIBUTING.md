# Contributing

## PR 受理基準 (厳格)

セル更新・追加 PR は以下を全て満たすこと:

1. **原文 URL** (深リンク推奨)
2. **節/段落番号** ("第3章 2-1" のような明示)
3. **取得日** (ISO 8601: `2026-04-20`)
4. **信頼度** (`high` / `medium` / `low`、判定根拠を PR コメントに)
5. `requirement` (`must` / `should` / `mention` / `none`) と原文の整合性

これらが欠けた PR は merge しません。

## ガイドライン追加 PR

新規ガイドラインを追加する場合:

1. `corpus/guidelines/<id>.yml` を作成 (`docs/SCHEMA.md` 参照)
2. 13 列全てを記載 (`none` でも明示)
3. `scripts/validate_corpus.py` を pass
4. README の対象ガイドライン一覧を更新

## 法的助言ではない

本資料は参照用コンパイルです。利用者の判断責任で使用してください。
セル内容の正確性は原文との整合性に限り保証します (法的解釈は保証しません)。

## CLA

CC-BY 4.0 で公開されることに同意した上で PR してください。
